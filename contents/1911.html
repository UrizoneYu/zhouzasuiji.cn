1911. 用户、内容、活动运营

Mar. 15, 2019

运营地图2.0 - 用户运营

1.数据获取流程 确定目标与需求，数据字段规划，埋点采集数据，数据报表制作

2.海盗模型 获取数据，激发活跃，提高留存，增加收入，病毒传播。

3.用户状态模型 不活跃用户：流失用户、不活跃用户。 活跃：新增用户，回流用户，活跃用户，忠诚用户。

4.用户真诚度模型与用户价值模型 用户价值分析模型(RFM模型）：R(Recency)表示用户购买的时间有多远，F(Frequency)表示用户在时间内购买的次数，M (Monetary)表示用户在时间内购买的金额。

5.用户分层与分群 分层：根据具体业务指标进行分层。 分群：根据地域、人口特征、场景等进行分群。

6.用户画像 你的用户是谁？你的用户喜欢什么？你的用户在哪里？你的用户流失率？

7.用户成长体系 新手期：提升成长速度 成长期：提升用户的忠诚度 成熟期：提升用户APPU 衰退期：延长用户生命周期 流失期：挽回流失用户

运营地图2.0 - 内容运营

1.内容运营基础流程 内容运营：通过内容的产生、加工、组织、呈现，从而提高互联网产品的内容服务价值，已达到产品拉新、留存、最终付费的运营目的。 基本流程：原生内容 -> 内容加工 -> 推荐推送 -> 用户 -（产生）> 原生内容（继续闭环）。

2.内容二次加工 由内到外为：选题，标题，配图，摘要；其中摘要写作的三个方式为：提炼核心思想，摘要文章金句，引用名人名言与段子。

3.基于用户生命周期的内容运营 新手期：通过新手引导方式让其活跃（如启动页内容引导，优质内容推荐，登录注册提醒等）。 成长期：分类推荐，搜索推荐，人工推荐等推荐方式提高活跃。 成熟期：通过拓展用户的兴趣范围、搭建内容成长体系等方式达到内容互动的目的。 衰退期：借助基于用户兴趣、热门信息等内容推荐，进行挽留。 流失期：通过分析用户行为数据，梳理流失愿意；并制定召回方案。

4.内容功能打造完整流程 产品定位 -> 产品拆解 -> PRD制作 -> 产品测试 -> 全量发布 -> 活动策划 -> 数据分析 -> 数据报表 -> 项目回报 -> 竞品分析

5.APP推送 推送文案：热点资讯要快 - 10字内表达主旨；飞人非热点资讯要稳 - 突出最关心需求。 推送时间：上班：7-9，午休,11半-13半，下班：5-7，睡前：9-11。 推送人群：基于行为信息、优惠信息、设备信息、用户信息等分类信息。

6.高打开率推文原则 文章要写给目标用户看，而标题要吸引潜在用户；关键词尽可能前置；多用疑问句。

运营地图2.0 - 活动运营

1.活动背景 产品数据，市场热点，竞品动态，目标人群，领导观点

2.活动目的 拉新目的，促活目的，留存目的，转化目的

3.TIP活动形式模型 工具+场景+包装

4.GRAI复盘法 目标+结果+分析+总结

5.推演复盘法 预期 -> 实际 -> 原因 项目背景？目标人群？项目目的？目标指标？关键实现环节？具体方案？风险与避免？
